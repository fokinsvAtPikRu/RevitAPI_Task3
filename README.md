# Описание работы плагина
Создается и запускается транзакция
Выбранные пользователем элементы помещаются в коллекцию ElementSet
В методе PreData происходит проверка корректности выбранных элементов:
 - должен быть выбран только один элемент AreaReinforcement
 - выбранный AreaReinforcement должен быть прямоугольным 

В случае не выполнения этих условий транзакция откатывается и плагин завершает работу с результатом Failed.
Далее для выбранного элемента в методе TurnOffLayers устанавливаются BuiltInParameter для выбранного элемента, видимо выключаются какие-то слои армирования.
После в методе ChangeHookType удаляются крюки армирования. 
Потом выводится оповещение пользователю что плагин успешно отработал.
